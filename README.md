# Student Management API

## Description

This project is a Student Management REST API built using Node.js and Express.js.

## Features

- Get All Students
- Add Student
- Update Student
- Delete Student

## Technologies Used

- Node.js
- Express.js
- Postman

## Installation

```bash
npm install
```

Run:

```bash
node app.js
```

## API Endpoints

GET /student

POST /student

PUT /student/:id

DELETE /student/:id
